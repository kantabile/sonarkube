Statement를 하나만 사용하는 람다를 표현하는 방법은 두 가지가 있지만, 더 잘읽히고 간결한 방법은 정해져있습니다.

**참고**: 이 규칙은 프로젝트의 sonar.java.source의 값이 8보다 낮은 경우(java8) 자동으로 비활성화됩니다.

### 규칙을 어긴 코드

```java
x -> {System.out.println(x+1);}
(a, b) -> { return a+b; }
```

### 규칙을 준수한 해결책

```java
x -> System.out.println(x+1)
(a, b) -> a+b    // return 문 같은 경우도 없앨 수 있습니다
```
